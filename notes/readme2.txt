A bit has been defined as one binary digit.
It is the smallest data in a computer system that can take either 0 or 1.

To find how much memory your computer has, go to task manager and go to memory.

To go to your hard disk drive, do windows + e and go to your c: drive. After that, go to properties and check out the capacity. 

An easy shortcut to get to your command prompt it windows + r and then type cmd. Also if you want to access the c: drive, you can easily to cd\. If you want to make a directory inside of the c: drive, you can do md "file name"
